# 🕉️ Mahashivratri - The Divine Journey

A stunning, immersive spiritual web experience celebrating Lord Shiva and the sacred 12 Jyotirlingas. Built with cutting-edge web technologies to create an unforgettable digital pilgrimage.

## ✨ Features

### 🎨 Visual Excellence
- **3D Cosmic Background**: Live Three.js scene with animated torus knot and particle effects
- **Custom Cursor**: Golden cursor follower with smooth physics
- **Glassmorphism UI**: Premium frosted glass effects throughout
- **Smooth Animations**: Powered by Framer Motion with spring physics
- **Responsive Design**: Beautiful on all devices

### 🙏 Spiritual Content
- **Bilingual Experience**: Complete Hindi (Devanagari) and English support
- **12 Jyotirlingas**: Detailed cards with images, locations, and spiritual significance
- **Sacred Stotram**: The complete Dvadasha Jyotirlinga Stotram in Sanskrit
- **Story of Shiva**: Interactive narrative sections (Adiyogi, Neelkanth, Nataraja)
- **Infinite Mantra Scroll**: Continuous "Om Namah Shivaya" marquee

### 🚀 Technology Stack
- **React 18** - Modern UI library
- **Vite** - Lightning-fast build tool
- **Three.js + R3F** - 3D graphics and animations
- **Framer Motion** - Advanced animation library
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Beautiful icons

## 🎯 Getting Started

### Prerequisites
- Node.js 16+ installed
- npm or yarn package manager

### Installation

1. **Install dependencies**:
```bash
npm install
```

2. **Start development server**:
```bash
npm run dev
```

3. **Open in browser**:
Navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
npm run preview
```
